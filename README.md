# Discrete-Event Simulation Projects

Two simulation studies evaluating operational performance and testing
data-driven system improvements.

## Overview

This repository contains two pair projects completed for the Simulation course
at the University of Edinburgh.

### Project 1: Ride-Sharing Operations

The first project models the fictional BoxCar ride-sharing service in
Squareshire. The study evaluates rider waiting times, driver utilisation, and
system throughput before testing operational changes intended to improve rider
satisfaction and driver productivity.

### Project 2: Laptop Manufacturing

The second project models a laptop manufacturing system. It identifies
bottlenecks, evaluates baseline performance, and tests changes to capacity and
process design.

## My Contribution

Across both projects, I contributed to:

- input-data analysis and distribution fitting;
- construction and evaluation of the baseline models;
- interpretation of simulation outputs;
- comparison of proposed operational improvements.

## Methodology

- Discrete-event simulation in SIMUL8
- Input distribution fitting and goodness-of-fit analysis
- Baseline-model validation
- Replication and confidence-interval analysis
- Scenario comparison and operational recommendations
