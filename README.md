# Simulation_Projects
Two pair projects completed as part of the "Simulation" course at the University of Edinburgh. 

Project 1 involved evaluating the operational performance of the fictional BoxCar’s ride-sharing app in Squareshire and proposing data-driven
improvements to increase rider satisfaction and driver productivity. 

Project 2 involved evaluating the performance of a laptop
manufacturing system and testing operational improvements. 

In both projects I contributed by carrying out input data analysis and Baseline Model analysis.
